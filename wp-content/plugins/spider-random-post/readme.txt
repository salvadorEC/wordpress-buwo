Spider Random Post allows you to show posts in a random order in a sidebar. You can display simultaneously as many random posts as you need.  

== Description ==
Display random posts on the sidebar from the same category. You can choose the number of posts and the different effects available.    
  
Spider [Random Post](https://web-dorado.com/products/wordpress-random-post.html)
[User Guide](https://web-dorado.com/wordpress-spider-random-post/installing.html) 
  
WordPress Random Post plugin is a small but very smart solution for your WordPress web site. It allows you to show posts in a random order in a sidebar. You can display simultaneously as many random posts as you need with chosen effect and delay time.

  
###Features of the plugin

* WordPres 3.0+;  
* Possibility to customize the number of randomly displayed articles, auto update and updating time  
* 4 different effects for randomly displaying the articles  


If you want to have possibility to choose the particular category of posts to display randomly, you can buy the commercial version [here](https://web-dorado.com/products/wordpress-random-post.html).
  

== Installation ==  
  
####Thank you for your interest in Spider [Random Post](https://web-dorado.com/products/wordpress-random-post.html).  
  
Minimum requirements:  
  
* Wordpress 3.0+  
* PHP 5.x  
* MySQL 5.x  
  
After downloading the ZIP file  
1.  Login to your WordPress site and head over the 'Plugins' menu    
2.  Click 'Add New'  
3.  Choose the 'Upload' option  
4.  Click **Choose file** (**Browse**) and select the Spider [Random Post](http://wordpress.org/extend/plugins/spider-random-post/) zip file.  
5.  Click **Install Now** button.  
6.  Once it is complete, activate the plugin.

== Frequently Asked Questions ==

= How is the random post widget configured? =  

You can add the widget to a sidebar the following way. Spider Random Post is available in Widgets under the menu item Appearance.

Here are the parameters of the widget.

  * Title. Fill the title for the widget on Spider Random Post   
  * Quantity of [Wordpress Random Posts](http://wordpress.org/extend/plugins/spider-random-post/). Show the number of random posts on Spider Random Post.  
  * Auto Update. Choose whether you want auto update or not on the Spider Random Post.  
  * Visualization. From the given list choose the style of displaying the random posts on Spider Random Post (visual effects).  
  * Time of update(sec). Define the period of the update of [Wordpress Random](http://wordpress.org/extend/plugins/spider-random-post/) posts on Spider Random Post.
